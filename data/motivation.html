Hello! My name is Keonhyeong Kim studying economics in South Korea.

This is a website for a personal project to share my data of Korean economy, especially financial ones for some factor models like the Single Index Model or Fama-French 3 Factor Model. When I was on bachelor course in economics, I tried to practice factor models using Korean Market Risk Premium, SMB, or HML data. But I found that there were no websites to easily get that kind of Korean ones. Even it was so hard to get enough PRICE data to be applied on the models.

So I decided to make a website for anyone to easily get those of data files available free of charge, benchmarking French's data library, and just this webiste is. I'm going to upload not only data files with the source codes, but also concise explains for the models. So don't trust me and please check out the source codes.
